# ウナギの迷宮

**Current Version:** v1.4.0-beta1

## 概要

ウナギを操作して迷路を進み、ゴールを目指すブラウザゲームです。

ゲームシステムの基盤は完成し、現在はウナギらしい自然な動きと描画表現の向上、さらに機能追加を見据えた設計改善を進めています。

## 現在の状況

- `class Game` を中心とした複数ファイル構成へリファクタリング済み
- Ver.1.0.1 と同等のゲームプレイを維持
- 一定距離追従アルゴリズムによるウナギのニョロニョロ動作を実装
- 頭部・胴体・尾端を独立した描画メソッドへ整理し、描画処理をリファクタリング
- 尾端を三角形で描画し、よりウナギらしいシルエットを実現

現在は **v1.4.0-beta1** を、今後の機能追加に向けた描画システムの基盤となるバージョンとして開発を進めています。

## 開発目標

- ウナギらしい自然な泳ぎ・シルエットの実現
- 機能追加しやすい設計の維持
- PC・スマートフォンの両方で快適に遊べるゲームを目指す

## 今後の予定

- 尾先（涙滴形など）の描画表現を検討
- ウナギの泳ぎや波打ち表現のさらなる改善
- ステージ追加
- タイムアタックなどのゲームモード追加
