# ウナギの迷宮

**Current Version:** v1.2.1-alpha3 Step1

## 概要

ウナギを操作して迷路を進み、ゴールを目指すブラウザゲームです。

現在はゲームシステムの基盤が完成し、ウナギの自然な動きや描画表現を改善しながら開発を進めています。

## 現在の状況

- `class Game` を中心とした複数ファイル構成へリファクタリング済み
- Ver.1.0.1 と同等のゲームプレイを維持
- 一定距離追従アルゴリズムによるウナギのニョロニョロ動作を実装
- 頭部・胴体・波打ち表現を改良中

現時点では **v1.2.1-alpha3 Step1** を描画面で最も安定したバージョンとして開発を進めています。

## 開発目標

- ウナギらしい自然な泳ぎ・シルエットの実現
- 機能追加しやすい設計の維持
- PC・スマートフォンの両方で快適に遊べるゲームを目指す

## 今後の予定

- **v1.2.1**：描画品質の改善
- **v1.3.0**：尾柄・尾先を含むウナギのシルエット改善
- **v1.4.0**：ステージ選択機能
- **v1.5.0**：タイムアタックモード
- **v2.0.0**：ゲーム内容の拡張・正式版を予定
